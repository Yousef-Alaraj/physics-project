# Physics 2 Engine 🚀

A dedicated workspace for learning "Scientist Python" and building a Physics 2 simulation project. 

The goal of this repository is to bypass heavy software engineering concepts and focus entirely on using Python as a high-powered calculator and visualization tool for physics concepts (like electromagnetism, optics, or circuits).

## 🛠️ Tech Stack
* **Python 3.x:** The core language.
* **NumPy:** For high-speed array math and vector calculations.
* **Matplotlib:** For generating 2D graphs and plots.
* **VPython:** For rendering 3D physics simulations.

## ⚙️ Environment Setup

1. Install Python from [python.org](https://www.python.org/downloads/).
2. Open your terminal and install the required physics and math libraries using pip:
