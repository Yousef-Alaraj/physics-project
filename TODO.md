# Python Learning & Project Roadmap

## Phase 1: Core Syntax Crash Course
*Objective: Translate existing programming logic into Python syntax.*
- [x] Install Python and an IDE (like VS Code).
- [x] Learn Python variable assignment and basic math operators.
- [x] Understand Python Lists and Dictionaries.
- [x] Practice control flow: write `for` loops, `while` loops, and `if/else` statements in Python.
- [x] Write a basic function (e.g., a function that takes mass and acceleration and returns force).

## Phase 2: The Physics Toolbelt (Math & 2D)
*Objective: Learn how to crunch numbers and graph them.*
- [x] **NumPy Basics:** Create a 1D array of numbers (representing time steps, e.g., $t = 0$ to $t = 10$).
- [x] **NumPy Math:** Apply a mathematical function (like sine or cosine) to an entire array at once.
- [x] **NumPy Vectors:** Learn how to add and multiply vectors (crucial for forces and fields).
- [x] **Matplotlib Basics:** Plot the NumPy time array on the X-axis and the math function results on the Y-axis.
- [x] **Matplotlib Styling:** Add a title, axis labels, and a legend to the graph.

## Phase 3: Project Execution
*Objective: Build the actual Physics 2 assignment.*
- [ ] Define the specific physics problem/scenario being modeled.
- [ ] Write out the physics formulas required on paper first.
- [ ] Create the core calculation engine in Python (using NumPy).
- [ ] Build the output (either Matplotlib graphs or a VPython 3D scene).
- [ ] Refactor and clean up the code for the final submission.
